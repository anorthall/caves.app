Import data

Data can be imported to the database from a CSV file. This can be useful to add a large number of entries at once. To begin, download the template file to use as a starting point.

The template file contains the column headings that are required for the import to work. Leave the header row intact and add your data below it. You must not change the order of the columns, but you can leave columns blank if you do not have data for them.

You can add a maximum of 50 trips in any one import. You are welcome to use the import function several times if you have more than 50 trips to add.

The following fields are required: cave name, trip start date/time, and trip type. All other fields are optional. The available trip types are: {{ trip_types }}.

Distances (vertical distance up, vertical distance down, horizontal distance, aid climbing distance, survey distance and resurvey distance) must be entered with the unit of measurement. For example, 10 metres would be entered as 10 m and 10 feet would be entered as 10 ft.

Dates and times work best in the format YYYY-MM-DD HH:MM and with the time in the 24 hour clock. For example, January 1, 2018 at 1:00 PM would be written as 2018-01-01 13:00. The easiest option to ensure this formatting is to get your spreadsheet program to format the cells appropriately for you.

Once you have uploaded the data, you will be able to review and edit it before it is added to the database.
