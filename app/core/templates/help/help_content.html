Help with caves.app

This website is in active development and sometimes things may not go as smoothly as we'd like.

If you need any help with anything, or if you've encountered a problem with the site, then please do not hesitate to send me an email and I will get back to you as soon as possible. Equally, any feedback — positive or negative — is very welcome.

There are also certain tasks I can help with, on request:

  • Import of a large amount of trips — if you have a large amount of trip data, perhaps stored in a spreadsheet, I'd be happy to help add it to the site.
  • Bulk editing — for example bulk deletions, or changing the formatting of many fields.
  • Account deletion — if you no longer wish to use the site.

The support email address is: admin@caves.app.
